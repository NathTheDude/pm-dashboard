# Project Overview

This project is a dashboard for project management that helps teams track their progress and manage tasks efficiently. The dashboard includes features like task assignment, progress tracking, and reporting.

## Setup Instructions

1. **Clone the Repository**
   ```bash
   git clone https://github.com/NathTheDude/pm-dashboard.git
   cd pm-dashboard
   ```

2. **Install Dependencies**
   Make sure you have Node.js installed. Then run:
   ```bash
   npm install
   ```

3. **Run the Application**
   ```bash
   npm start
   ```

4. **Access the Dashboard**
   Open your browser and go to `http://localhost:3000` to access the dashboard.

5. **Configuration**
   Update the configuration files as needed for your environment.
